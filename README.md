# Landon's Fitness Meals - Custom WordPress Theme

### Project Overview
This is a custom e-commerce theme built for a fitness meal delivery service. It utilizes a custom theme architecture and deep WooCommerce integration.

### Implemented Features
- **Custom Post Type:** "Meal Reviews" for customer testimonials.
- **Shortcodes:** Custom `[fitness_promo]` banner for site-wide sales.
- **WooCommerce Overrides:** Custom `single-product.php` template for a unique meal detail view.
- **Front Page:** A dedicated `front-page.php` with a dynamic featured products grid.

*Note: Due to local XAMPP database connection issues, the live site is not available but all logic and template hierarchy have been fully implemented in this repository.*
